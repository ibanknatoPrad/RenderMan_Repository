FGHexaTile_bump.sl

Simple displacement shader that creates a Hexagon pattern using Voronoi/Worley noise. This shader contains a tweak version of the ARMan voronoi_2d_f1f2() call.

NOTE: This shader makes use of the following Advanced RenderMan include files: project.h and patterns.h.
