RudyCstripeCeramic.sl

Ceramic with feeling that it was painted while rotating on the molding table. This shader needs includes found in the shaderDev.tgz (More information can be found at: RMR Shader dev). It also uses Rudy's rclocillum.h file.

Also, take note, the RIB example needs an environment map. It currently is using Ivan's funcenter image. To turn it into a texture map for PRMan, I did: "txmake -mode clamp -latlong funcenter.jpg funcenter.tx"
